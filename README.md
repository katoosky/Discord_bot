# Discord_bot
かとうさんの運用しているBot

機能を追加したい場合は、以下の通りにお願いします。

- リポジトリをフォークする
- フォークしたリポジトリで、Cogを作成し、Bot2のcogsディレクトリに配置
- Bot2のINITIAL_COGSにcogを追加
- Pull Requestを作成する

Pull Requestがきたらこちらで確認して問題なければマージします。
